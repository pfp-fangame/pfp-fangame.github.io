Introduction to the Manual

Last updated: 19th March, 2023

Welcome to the official PFP manual for everything pillar-related!

This manual compiles everything you need as a pillar builder for PFP pillars such as current pillar slots, kit contraptions, game standards, monuments, submissions, usage of the kit, and a hella-lot more.

If you happen to find inaccurate information while browsing, feel free to contribute at the manuals GitHub repository. The README located has information on how to commit changes at GitHub and contributing to the repository.
